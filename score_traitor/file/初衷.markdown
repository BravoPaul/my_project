## 高考志愿到底该怎么填报，让你避免90%的家长都不知道的填报大坑
### 填报志愿要先端正态度
很多人说填志愿是12年学海生涯的最后一步，一定要认真对待，避免折戟沉沙，留下遗憾。但其实我不太认同这种观点，毋庸置疑，填志愿是非常重要的，但是它绝对不是终点，他恰恰是一个人职业教育的起点。因为高中毕业那一刻就意味着我们基础教育结束，填志愿是我们在选择自己未来的职业，甚至未来的生活。我说这个的目的是纠正很多同学报志愿的错误观点，很多同学甚至所谓的报考专家都认为报志愿最好的结果是能压线投档，不浪费一分，这才是对高中夜以继日努力的交代。持这样观点去报志愿很容易坑了自己，去到一个不喜欢的学校或者学了不适合自己又没有前途的专业，这样报志愿更有机会最大化的利用高考分数，但是却在自己未来职业之路上越走越难。大学教育本质上是职业教育，大学能教会我们一种技能，然后利用这种技能进入社会的某个行业发光发热，填报志愿其实就是我们工作的起点，甚至说是最重要的起点，所以我们要抱着找工作的心态去填报志愿，志愿填对了，至少你的职业道路路走对了，即使走的慢也会进步，最怕的就是在错误的路上越走越远。下面是分享给大家的志愿填报干货：

### 填报流程
- 了解感兴趣或者热门的行业：
	这步其实是广大家长和考生做的最差的一步，很多同学和家长根本不去认真了解一个行业，对行业的认识基本来自于非行业内的人士的道听途说，同时互联网上有太多三观不正、不负责任口嗨的内容，用这些内容去判断一个行业能准么？甚至有些专科生冒充985学生给准985同学报志愿，根本不懂计算机的给报考计算机的学生指点江山，这种人真是拿别人的前途当儿戏，可是现实中很多家长和同学就是拿着这些不专业不负责任的内容去想象出一个行业的图景，然后判断自己是否喜欢这个行业、以后是否要从事这个行业，这也难怪相当比例的学生不喜欢自己的专业，甚至有回去复读的同学。现在的社会是一个高度分工的社会，不是内行人根本不了解行业真实的状况，他们看到的都是行业的表象。那广大家长和同学该怎样正确的了解一个行业，我把这个问题分成俩个方面：1，你该去了解一个行业的哪些东西（What）；2，通过哪些途径去了解（How）。
	- What: 想大概了解一个行业，需要关注行业的发展和行业内的工作。下面我来具体解释下：
		- 行业本身的状况：当你处在某个行业时，这个行业本身的发展决定了的高度，正如雷军说站在风口猪都会飞起来，你如果在一个好的行业，即便不怎么努力也会有一份不错的收入，例如计算机，相反，在一个差的行业，再怎么努力，也生活很艰难。想要搞清楚行业本身的现状，需要搞清楚俩个问题：行业目前发展怎么样？行业未来的发展？搞清楚第一个问题相对简单，也是最重要的，一定不要去学一个目前发展不好的行业，行业发展不好包括行业内人大量失业、转行、薪资低于社会平均水平等。行业未来的发展判断相对比较困难，如果大量的行业内的人都对该行业判断比较悲观，大家就别考虑了，虽然有可能大家的判断都是错的，但概率毕竟不大，我们何必把自己的前途压在这种小概率事件上呢？
		- 行业内的工作：同一个行业，不同的工种工作内容和晋升前途是千差万别的，拿IT行业来举例，人工智能工程师和测试工程师无论是薪资水平还是工作内容相差极大，同样资历的工程师，从事人工智能相关工作可能比测试工作工资高出3倍，而且一般来说未来的前途也比后者好的多。所以仅了解一个行业是不够的，还需要了解这个行业下不同工作的具体内容。具体来说，我们可以抽象出3个问题：行业下有哪些工种？每个工种具体的工作是什么？每个工种的前途怎么样？第1，2个问题了解起来相对容易，最后一个问题其实谁也说不准，但是大家可以从侧面了解，简单来说，就是看这个行业下哪些工作的人的薪资是最高的，哪些工作的人才最近几年需求量增速最快。资本一般是不会骗人的，某种工作工资远高于其他工作一定是有他的道理的。

	- How ：这部分我们介绍下，该通过哪些途径了解一个行业的状况
		- 最有效的办法就是找到从事这个行业的人去了解，但是一定要注意这个人三方面的背景：入行时间、教育背景、行业经历。对于入行时间，最好是已经从事某行业3年以上的人，因为时间太短他自己也不能很深入的了解一个行业，讲出来的东西可能比较主观，这里注意下，很多人说，最好了解某个行业的方式是问问学这方面的学长学姐，其实不是的，他们只能告诉你：学校要学哪些知识，专业课有哪些，学校坑不坑等学习相关的事情，其实他们对于行业的判断也比较粗浅，因为你学某方面的知识和从事某方面的工作是两码事，不在一个行业摸爬滚打几年是很难了解一个行业的全貌的；教育背景也非常重要，举个例子，斯坦福大学毕业的人工智能博士和国内普通2本毕业的人工智能专业的本科生对于行业的认识一定是不一样的，普通学校的本科生肯定会告诉你人工智能这个专业很难找工作，大家都学这个，竞争压力太大。但是斯坦福的博士肯定会告诉你人工智能的人才是行业内稀缺的人才，无论是企业还是高校都非常好就业，待遇也很好。所以你要注意咨询对象的教育背景，并不是教育背景越好的人说的东西越权威，要注意你和他的教育背景的差别，很多行业是赢者通吃的行业，优秀的人才待遇极好，差点的工作都找不着，大部分只在自己的立场考虑问题，这样就会无意误导了教育背景差别很大的同学；最后要关注的是行业经历，有的人从事一个行业很久了，但是每天做的事情很固定，打交道的也就和他有业务往来的同事，那么他的行业经历其实是不丰富的，所以咨询的对象最好是跳过槽或者在行业核心部门的人，他们对行业的认识相对更深刻。
		- 直接找到愿意帮助你的行业从业者相对比较难，第二种方式就是到网上搜，大家搜的时候要注意不要去看新闻类的文章，这些一般是为了博人眼球，增加阅读量的，没有任何干货，建议知乎这样的平台，里面有很多问题如：从事xxx是一种什么样的体验，不少行业内的人回答的蛮客观的，因为在互联网上去判断回答者的职业经历比较困难，建议大家多看，多总结，不要听信一家之言。

- 选择学校和专业
当你大概心中有一个从事的行业的目标时，就可以筛选学校和专业了，到这里有个很经典的问题：学校、城市和专业究竟哪个更重要？这里有个比较简单的办法：医学、师范类城市相对更重要，这些专业大部分都是在哪学在哪就业；理工科类专业排名更重要，因为理工科到哪里都是一样的，北京的技术和纽约的技术用的大同小异，所以专业技术能力很重要；文科金融学校很重要，因为这些学科没有统一答案，学校能给你提供的资源就很重要了。当然上述比较都是建立在差距在一个级别以内的高校之间的对比，清华最差的学科绝对比普通211最好的学科好，这样对比就没有意义了。下面我就介绍一下具体的选择学校和专业的方法：
	- 根据位次判断自己报考的大学能到第几梯队，下面是我对中国大学的层级判断：第一梯队：清华、北大，第二梯队：华五（浙江大学、复旦大学、上海交通大学、南京大学和中国科学技术大学）+人大，第三梯队：其他985院校+重点211院校，第四梯队：其他211院校，第五梯队：一本非211院校，第六梯队：二本，第七梯队：三本及专科。文章的后面会详细的列出各大学及其梯队。这里肯定很多同学会觉得有些排名不合理，比如第三梯队里分数较高的哈尔滨工业大学和分数较低的重庆大学怎么能是同一个级别呢？可能你们班第一名能考哈工大，第五名考重庆大学，对于学生来说差别确实很大，但其实在企业招聘的角度来看，哈工大和重庆大学差别还真的不大（我这里指的是俩个大学的优势学科，拿哈工大的优势学科和重大的弱势学科比没有意义），俩个大学的毕业生都是棒棒哒。因为高考是个很有运气成分的事情，发挥好与坏导致的差十几甚至二十几分是很正常的事情，难道你们班第一没有发挥失常考了第五的情况么？所以这里的梯度大家可以看做是一个基本的标准，在一个标准里的学校基本你们能得到的资源是相似的，也就是说考验、出国、就业得到的机会不会有质的差别。
	- 如果你想报考的专业为理工科，那么你需要依次参考同梯队学校专业排名、学校排名和城市排名。我们来举个例子：假如2019年河北某考生分数为659分，希望报考计算机相关专业，这个分数全省排名为1436名，清北肯定不用想，华五里面分数稍低的南京大学在河北录取位次大概在400名-500名，所以华五也凉凉。我们只能从第三梯队学校里选，梯队确定了以后，我们这个时候就要看同梯队大学专业排名，因为这个梯队大学比较多，我们只拿电子科大和武汉大学举例子，电子科技大学计算机全国排名第11，2018年全省2500名可以上电子科大的计算机系，而武汉大学计算机排名全国第18，全省1200名可以上武汉大学的计算机系，对比来看电子科大专业排名略强，并且分数也较低；我们再来考虑下学校综合实力，武汉大学排名大概20名左右，电子科大40名左右，武汉大学综合实力更强；最后城市方面，武汉和成都都是新一线城市，发展相似，基本可以打平。这里很多同学就会这样想，虽然专业排名电子科大略强，但是综合实力武大要强得多，况且武大的录取分数也高，我的高考分数是659分，假如第一志愿报考了电子科大的计算机系，岂不是浪费分数了么？所以我第一志愿要报考武汉大学计算机系作为冲一冲的学校，然后第二志愿报考电子科大作为稳妥的学校。我知道屏幕前80%的同学都是这么想的，但是，这种报考方式大错特错，这种报考方式大错特错，这种报考方式大错特错，重要的事情说三遍。因为假如你选了武汉大学，根据2019年录取结果来看，你是上不了武汉大学计算机系的，有的同学说我的第二志愿可以报电子信息类，根据2019武汉大学在河北录取结果，我是可以上电子信息类的，不同样美滋滋么？反正电子信息和计算机差不多。我可以很负责任的告诉你：电子信息类和计算机类差多了，学习的专业课差别很大，更重要的是当你以后找工作的时候，企业是非常重视你是不是科班出身这个事情的，作为一个面试过上百人的面试官告诉你，不是计算机科班出身，如果想去阿里、百度、腾讯这些大公司，基本你连一面都过不了。那这种情况该怎么填报呢？其实很简单点，直接报考电子科大计算机系就好，专业排名那么高，被调到第二专业的概率又低，成都的妹子还漂亮，岂不是美滋滋？不要觉得浪费了分，我们的高考分数是为了让我们能够选到合适大学的合适专业，这个目的达到就够了，至于那些多出来的分数，只能说明自己实力太强，等到上了大学，吊打压线进来的同学，再拿个国家奖学金，保送清北，不香么？
	- 但是如果你想报考文科类专业，那就不一样了，此时一定要关注学校排名，因为很多文科专业的同学最后的工作和专业关系不是很大，比如公共管理这种专业，以后的就业方向可能是公司的行政岗位、秘书、城市规划岗位等，这些岗位的工作内容和你大学学的专业其实没有很大的关系，所以公司对这些岗位的招聘不会看你专业排名，却比较重视学校排名，因为理论上越难考的学校出来的学生说明高中成绩越好，成绩好的学生一般做事也比较靠谱。报考这类专业的同学，往知名度高、综合排名好的学校冲就对了。对于师范类和医学类的专业，大家一定要多关注城市排名，大家有没有发现当地的老师很多都是当地或者省会城市某师大毕业的，这些专业的地域性是特别强的，基本在哪里读大学就在哪里就业，所以要看好城市，以后想去哪里工作就尽量报哪里的学校。
